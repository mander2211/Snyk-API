# SnykAPI


To Run Any py Script:
-Need Node.js installed
    https://nodejs.org/en

-Needed modules to run js scripts
    node-fetch
    prompt (future enhancment)
    -Run modules installed via terminal 
        "npm install {module}"

-Run JS via terminal
    node {script}
